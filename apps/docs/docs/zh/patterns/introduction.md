# 开发范式 (Development Patterns)

欢迎来到开发范式指南。AlienForm 是一个高度灵活的框架，这意味着通常有多种方法可以解决同一个问题。

为了防止产生“骇客代码（Hacker Code）”并确保大型团队之间的一致性，本指南概述了处理常见企业级表单场景的**标准化方法**。

通过遵循这些范式，你的代码将：

- 更容易被其他开发者阅读和维护。
- 性能更高（利用核心模型驱动，而不是依赖 React 的重新渲染）。
- 完全契合 AlienForm 的架构理念。

请阅读侧边栏中具体的范式文章，学习如何处理权限校验、形态切换、异步数据、编辑态初始化、复杂组件建模，以及规格与 SKU 销售矩阵这类更复杂的企业级场景。

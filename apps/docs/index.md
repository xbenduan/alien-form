  - title: Pure JSON Schema
    details: Fully implements the Formily Schema Protocol with x-* extensions. Define forms entirely in JSON.
  - title: Declarative Linkage
    details: Active and passive field linkage modes with an expression engine supporting $self, $form, $values, $deps, $target.
  - title: Array Fields
    details: First-class support for repeatable fields — ArrayCards and ArrayTable with push/remove/moveUp/moveDown.
  - title: Async Data Sources
    details: Load remote options with x-async-data-source. Supports dependency-based cascading and auto-reload.
  - title: Extensible Components
    details: Register custom components and decorators. ReadPretty mode support for display-only variants.
